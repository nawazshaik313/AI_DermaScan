# DermalScan Elite

AI-powered dermatological diagnostic system using deep learning.

## Features
- Face detection using OpenCV DNN
- Skin condition classification
- Live camera analysis
- PDF medical report generation
- CSV analytics logging
- Confidence heatmap visualization

## How to Run
1. Install dependencies
   pip install -r requirements.txt
2. Run app
   streamlit run app.py

## Supported Inputs
- Image upload
- Camera capture
- Live streaming (mobile & laptop)

## Disclaimer
This system is for educational purposes only and not a medical diagnosis tool.
